# CodeAlpha Data Visualization

## Objective

Create visual representations of the Titanic dataset to uncover patterns and insights.

## Tools Used

* Python
* Matplotlib
* Seaborn
* Jupyter Notebook

## Visualizations

* Survival Distribution
* Gender Distribution
* Passenger Class Distribution
* Age Histogram
* Fare Distribution
* Box Plot
* Correlation Heatmap
* Scatter Plot

## Outcome

Visualizations helped understand passenger characteristics and survival trends effectively.
